# ChadDa-server
위치, 사용자, 게시글 기반 대용량 데이터 검색 서비스 ChadDa입니다.

<br>

### Repository 설명

모든 비즈니스 로직이 들어있는 Spring Application입니다.

<br>

### 구현 기능

1. 회원 관련
   - 회원가입
   - 로그인
   - 로그아웃

2. 검색
   - 위치 검색
   - 게시글 검색
   - 유저 검색

3. 상세 검색/조회
   - 위치 정보가 포함된 게시글 검색
   - 게시글을 작성한 유저의 프로필 조회
   - 유저가 작성한 게시글들 조회
